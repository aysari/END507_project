Kodu Çalıştırma için Yapılması Gerekenler
	1) END507_project dosyası .zip uzantılı indirilir.
	2) END507_project .zip dosyasının içindeki END507_project  dosyası ECLIPSE'in directory'sine kaydedilir.
	3) ECLIPSE programı çalıştırılır.
	4) File -> Open Project From File System tıklanır.
	5) Package Explorer sekmesinden END507_project  -> src -> END507_project açılır.
	6) https://www.youtube.com/watch?v=sf59_7r8QSY&t=336s videosunun 03:17'inci saniyesinden itibaren anlatılan şekilde CPLEX librarysi  ilgili project'e eklenir. (Buradaki teknik detayları tam hakim olmadığım için YouTube videosundan faydalandım)
	7) Kod runlanır.
